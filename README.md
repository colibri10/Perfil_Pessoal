### Perfil Pessoal – Grupo 6



Um aplicativo Android simples e intuitivo desenvolvido para exibir informações básicas de um usuário, como nome, idade, profissão e foto, utilizando um layout construído em LinearLayout.

O projeto foi criado para fins acadêmicos, seguindo os requisitos solicitados pelo professor, com adição de funcionalidades extras para melhorar a experiência do usuário.



### Funcionalidades Implementadas



##### Funcionalidades obrigatórias



* Exibição de imagem com ImageView
* Exibição de nome, idade e profissão com TextView
* Botão Editar (inicialmente sem funcionalidade)
* Layout baseado em LinearLayout



##### Funcionalidades extras implementadas



* O botão Editar realmente funciona
* Troca de tela entre Perfil e Edição
* Modificação de informações do usuário
* A tela inicial exibe um nome pré-registrado, que pode ser alterado
* Retorno das informações atualizadas para a tela principal
* Todos os botões funcionais e navegáveis



### Tecnologias Utilizadas



* Android Studio
* Kotlin
* XML para layouts
* Android Jetpack (Intents para navegação)
* LinearLayout
